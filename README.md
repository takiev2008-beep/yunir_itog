# Дневник погоды

Простое приложение на Python с графическим интерфейсом для записи, фильтрации и хранения данных о погоде.

## Описание
Это приложение позволяет:
- Вводить новые записи о погоде (дата, температура, описание, наличие осадков)
- Просматривать все записи в таблице
- Фильтровать записи по дате и температуре
- Сохранять записи в файл `weather.json`
- Загружать сохранённые записи при запуске программы

## Как запустить
1. Убедитесь, что у вас установлен Python 3.x.
2. Скачайте файлы `main.py` и `weather.json` (последний можно оставить пустым или использовать начальные данные).
3. Запустите `main.py` командой:
```bash
python main.py
